Essential commands

Everything you need in simple, memorable commands

queryly connect add

Add a new database connection with interactive prompts

queryly connect list

Show all saved connections with last used dates

queryly schema list <conn>

List all tables with row counts and sizes

queryly schema info <conn> <table>

View detailed table structure and column information

queryly data browse <conn> <table>

Interactively browse table data with pagination

queryly data query <conn>

Enter interactive SQL query mode with formatted results

queryly data export <conn> <table> csv

Export table data to CSV or JSON format

queryly connect test <name>

Test if a saved connection is working properly

üíª Windows Installation

1

Download and Extract

Download queryly-windows.zip and extract it to your preferred location

2

Run the Install Script

Open PowerShell as Administrator in the extracted folder and run:

.\install.ps1

This adds Queryly to your PATH so you can run it from anywhere

3

Test with Sample Database

Run the test script to create a sample SQLite database:

.\create-test-db.ps1

Creates a test database with sample data to explore

4

Start Using Queryly

queryly connect add # Follow the prompts to connect to the test database # Connection string: Data Source=./sample-db.sqlite
To uninstall: Run .\uninstall.ps1 in PowerShell as Administrator

üêß üçé Linux / macOS Installation

1

Download and Extract

Download the appropriate .tar.gz file and extract:

# For Linux tar -xzf queryly-linux.tar.gz cd queryly
# For macOS tar -xzf queryly-macos.tar.gz cd queryly
2

Make Scripts Executable

Give execute permissions to the installation scripts:

chmod +x install.sh uninstall.sh create-test-db.sh
3

Run the Install Script

Install Queryly system-wide:

sudo ./install.sh

This copies Queryly to /usr/local/bin and makes it available globally

4

Test with Sample Database

Create a test database with sample data:

./create-test-db.sh

Generates test.db with users, orders, and products tables

5

Start Using Queryly

queryly connect add # Connection string: Data Source=./sample-db.sqlite
To uninstall: Run sudo ./uninstall.sh

üîß Build from Source (Developers)

1

Clone the Repository

git clone https://github.com/Ayine-nongre/queryly.git cd queryly
2

Build the Project

Requires .NET 8 SDK or later

dotnet build
3

Run Queryly

dotnet run --project src/Queryly.CLI/Queryly.CLI.csproj -- connect add

Optional: Create an Alias

# Add to ~/.bashrc or ~/.zshrc alias queryly='dotnet run --project /path/to/queryly/src/Queryly.CLI/Queryly.CLI.csproj --'

üöÄ Quick Test After Installation
